Introduction to C++

C++ is one of the most important modern programming languages, enabling both low-level memory control and high-level abstractions. It is widely used for system programming, high-frequency trading, games, embedded systems, large-scale applications, and performance-critical tasks.

  • Combines power (C) with abstraction and OOP features.
  • Compiled, fast, and portable.
  • Standard Template Library for fast, generic containers and algorithms.
  • Supports direct low-level memory control and high-level abstractions.
  • Used by leading companies (Microsoft, Adobe, game studios, Google, Facebook...)

C++ Basic Syntax

The C++ syntax closely resembles C, but adds many new keywords and features. Standard libraries are included with the #include directive. The main() function is the program's entry point. C++ uses cout and cin for output and input (from iostream).

  • Uses #include to import libraries. iostream allows input and output.
  • using namespace std; simplifies code by not needing std:: everywhere.
  • Every program starts with main().
  • Statements end with a semicolon ;. Curly braces define code blocks.
#include <iostream>
using namespace std;

int main() {
    cout << "Hello, World!";
    return 0;
}

Data Types in C++

C++ supports all primitive types from C, and adds many rich types for objects and containers. Understanding types is crucial for writing efficient and correct code.

  • int: Whole numbers (e.g., 5, -2, 789)
  • float/double: Decimal numbers, single/double precision
  • char: Single ASCII character (e.g., 'A')
  • string: Text (#include <string>)
  • bool: Boolean logic (true or false)
  • unsigned: No negatives (e.g., unsigned int)
int age = 21;
double price = 19.99;
char grade = 'B';
bool isOpen = true;
string name = "Sam";

Control Flow in C++

Use conditional statements and loops to direct your program's logic. C++ supports nested and combined controls—vital for building flexible programs.

  • if, else if, and else drive decision-making.
  • for, while, and do...while provide loops for repeated actions.
  • switch selects between multiple constant options.
int x = 10;
if (x > 5) {
    cout << "x is greater than 5";
} else {
    cout << "x is 5 or less";
}
            
for (int i = 0; i < 3; i++) {
    cout << i << " ";
}

Arrays & Strings in C++

Arrays store a fixed number of items of the same type; std::string supports powerful, dynamic strings. For flexible lists with changing size, use std::vector from the STL.

  • Arrays hold fixed-size sequential collections of the same type.
  • Strings use std::string, allowing dynamic length and many functions.
  • Iterate using loops.
int a[4] = {1,2,3,4};
cout << a[2]; // prints 3

#include <string>
string s = "Welcome!";
printf("%zu", s.length());

Functions in C++

Functions enable code re-use and logical separation. C++ allows default arguments, function overloading, and inline functions. Functions improve maintainability and can pass objects by value, reference, or pointer.

  • Reusable blocks of code; can return a value and/or take parameters.
  • Arguments are passed by value (copy) or reference (with &).
  • Use void for functions without return values.
int add(int a, int b) {
    return a + b;
}

int main() {
    cout << add(5, 3);
    return 0;
}

Introduction to OOP (Object-Oriented Programming)

OOP in C++ is based on classes, which combine data and methods. Principles like encapsulation, inheritance, polymorphism, and abstraction make large programs easier to write and maintain. Put each class definition in its own header/source files for large projects.

  • C++ is famous for supporting OOP: organizing code around objects/classes.
  • Classes bundle data (members) and behavior (methods).
  • OOP enables encapsulation, inheritance, and polymorphism.
class Car {
public:
    string brand;
    int year;
    void honk() {
        cout << "Beep!\n";
    }
};

int main() {
    Car myCar;
    myCar.brand = "Toyota";
    myCar.year = 2022;
    myCar.honk();
    cout << myCar.brand;
    return 0;
}

C++ STL (Standard Template Library)

The STL (see cppreference STL overview) includes generic containers, algorithm functions, and iterators. STL dramatically increases productivity and performance for common tasks like sorting, searching, and data management.

  • Provides efficient, ready-made classes for collections and algorithms.
  • Common STL containers: vector, map, set, queue, stack.
  • Containers are generic (work on any type using templates).
#include <vector>
#include <algorithm>
vector<int> nums = {4,1,3};
sort(nums.begin(), nums.end());
for (int n : nums) cout << n << ' ';

Inheritance in C++

Inheritance lets you define new classes using existing classes as a base. Supports single, multiple, and virtual inheritance (to prevent diamond problem). It's a core pillar of OOP design and enables code reuse and hierarchy modeling.

  • Allows new classes to acquire properties/methods of existing ones.
  • Supports reusability and extension.
  • Base class: parent | Derived class: child.
class Animal {
public:
    void speak() { cout << "Generic sound\n"; }
};

class Dog : public Animal {
};

int main() {
    Dog d;
    d.speak(); // Outputs: Generic sound
    return 0;
}

Polymorphism in C++

With polymorphism, functions or methods can behave differently based on the context or the object's type. Virtual functions enable dynamic method dispatch. Learn about virtual and override.

  • Achieved using function overriding and virtual functions.
  • Allows child classes to define specific implementations.
class Animal {
public:
    virtual void speak() { cout << "Animal sound\n"; }
};
class Cat : public Animal {
public:
    void speak() override { cout << "Meow\n"; }
};
int main() {
    Animal* a = new Cat();
    a->speak(); // Outputs: Meow
    delete a;
    return 0;
}
